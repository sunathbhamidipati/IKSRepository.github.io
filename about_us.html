We are a team of 5 students pursuing our final year of engineering from Mahindra University, coming from various backgrounds such computer science, artificial intelligence and electronics.

Our project is titled “Preserving Indian Knowledge Systems Using AI”. It started off as an open ended project because the concept of IKS is relatively new. After researching the same we decided to create a repository for all the resources related to different fields in this domain that could be used to aid any prospective research.

We were inspired by our government's IKS website to create this website as a one stop place for all the data related to IKS or any research conducted revolving around in India. We have built a repository that can be sorted by category which can be used to view research conducted in all these fields in various formats (research papers, journals or manuscripts).
